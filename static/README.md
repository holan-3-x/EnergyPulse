
# EnergyPulse Grid Management Frontend

This is the React frontend for the EnergyPulse system. It interacts with the Go-based API Gateway to visualize smart meter data, manage households, and facilitate energy price predictions.

## Run Locally

**Prerequisites:**  Node.js (v18+)

1. Install dependencies:
   `npm install`

2. Run the app:
   `npm run dev`

The application will start at http://localhost:3000.
Ensure the backend API Gateway is running on http://localhost:8080.
