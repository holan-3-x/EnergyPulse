QUICK SUMMARY (What You're Building)
================================================================================

A web application that:
1. Lets users register with their house details
2. Simulates 20 smart meters publishing energy data via MQTT
3. Uses ML to predict hourly electricity prices
4. Logs predictions on Ethereum blockchain (immutable proof)
5. Shows real-time predictions on a React dashboard
